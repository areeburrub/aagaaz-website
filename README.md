## To Run Locally
To Run this application locally your sytem should have Node Js installed so that you can install required dependencies

### Clone Repo
```git
git clone https://github.com/areeburrub/aagaaz-website.git
```

### After Cloning
```
cd aagaz-website
npm i
npm run dev
```
